Gregg Orr Auto

  • Website: www.greggorrauto.com
  • Status: Inactive

A very challenging (i.e., fun) client site that I built. The client wanted several custom features that we hadn't yet built for any other clients. Therefore, I had the opportunity to code some things from scratch.

The biggest challenge was the New and Used mega dropdowns that appear when you hover over their parent links in the navigation. The code to pull the vehicles from the database was an already built feature, but I had to tweak it slightly to separate the vehicle makes for New vehicles into tabs.

Due to the size of the images and the amount of database calls that would be required to show all of the vehicles for every make, loading every make on inital site load would be a huge performance hit to both the user's load times and for our servers. Therefore, I used AJAX to load each make individually when clicked on and killed the process for the previous make to keep page responsiveness optimal. Additionly, to improve the load speed of makes that had already been loaded, the data for them was cached for quick recalls.

While the background for this site looks amazing, getting the design to play well with all of the pages on the site was a challenge. The background in the initial Photoshop mockup worked fine for the home page and other pages similar in height, but for it to work on very tall pages like our inventory search and vehicle detail pages, the image for a repeating background had to be cut in very exact detail. The HTML code structure also had to be altered significantly compared to most client sites to implement the footer's design to the designer's satisfaction.
